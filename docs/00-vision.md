# PCTO Platform – Vision

Costruire una piattaforma SaaS multi-tenant per la gestione completa dei progetti PCTO delle scuole.

Obiettivi:
- Avviare un progetto PCTO in meno di 10 minuti
- Eliminare registri cartacei e firme manuali
- Automatizzare presenze, documenti ed export finale
- Offrire un catalogo di percorsi pronti gestiti dalla piattaforma

Utenti:
- Admin piattaforma
- Scuole (admin + tutor)
- Fornitori/aziende
- Studenti

Principi:
- Multi-tenant (School come tenant)
- Auditabilità completa
- Export PDF ufficiali con branding scuola
- MVP semplice > feature complesse

Non obiettivi:
- Firma digitale legale avanzata (V1)
- Integrazioni ministeriali complesse
